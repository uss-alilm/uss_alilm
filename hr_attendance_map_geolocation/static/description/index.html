HR Attendance map geolocation

With this module the geolocation of the employer is tracked at the check-in/check-out step, and manger can even restrict the attendance range for the employee.

Key Features

Employee Check In and Check Out location Google Map Link.
Set employees Active Work Location.
Set Attendance Range in meter.
Restrict employee to check in/out within Active Work Location Range.
