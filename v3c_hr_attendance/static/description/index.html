    How to Login?

    Open Server URL Screen by default when you open the mobile application
    1. Get "server details" & "your credential" from your server administrator / odoo administrator.
    2. Enter your odoo server URL.

      3. Here example of URL"https://runbot.odoo.com"

    3. Enter your port number. i.e. 8069 or "blank".
    4. Enter database name which database you want to login.
      i.e. "24070692-15-0-all"
    Click on Next button
    Now enter user details
    1. Enter your user name (same user name which you logn in to odoo backend).

      2. Here example of odoorunbot "admin"

    2. Enter your password (same password which you logn in to odoo backend).

      3. Here example of odoorunbot password "admin"

    3. If you want to rembember login details then click rembember me check box. (It is optional)
    Click to Login button

    How it works

    Dashboard Screen
    1. Allow Location permisison popup.
    2. Slide to check in. (Slide right to check-in).
    3. Slide to check out. (Slide left to check-out).
    4. Bottom of Slide button you can check last 5 history of check-in and check-out.

    Track Attendance from odoo backend

    APK File (Android Mobile Application)

    How to get APK file?
    1. Open "v3c_hr_attendance" module folder.
    2. Open "static" folder.
    3. Open "APK" folder.
    4. Now you copy "V3C_Attendance.apk" from this location and install it on your android device.
