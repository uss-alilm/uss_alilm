Journal Sequence

This plugin makes it simple to customise and modify the invoice, bill, credit, and debit notes sequence number from the journals.

Key Highlights

  • This plugin makes it simple to customise and modify the invoice, bill, credit, and debit notes sequence number from the journals.
  • The user can quickly set up the invoice, bill, credit, and debit note sequence from journals using the same easy approach as Odoo V13.
